# Hustle Print Hub

Official print order portal for HustleFanz. Banners, signs, decals, apparel, and promo materials for hustlers, creators, and businesses.

## Run the Project

**Run from the project directory (folder containing `package.json`):**

```bash
cd ~/Desktop/hustle-print-hub
npm install
npm run dev
```

The site runs at **http://localhost:5173**.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- React Router

## Pages & Routes

- `/` — Home (hero, product grid, how it works, sponsor section)
- `/order` — 8-step order wizard
- `/sponsor` — Print sponsor program, benefits, who it helps
- `/thank-you` — Order received confirmation

## Placeholders (No Backend Yet)

- File uploads
